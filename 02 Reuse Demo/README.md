# Reuse Demo

"Understanding how DITA supports modular documentation through Content Reuse."

## Project Summary

This project demonstrates how DITA enables modular documentation through content reuse. I used conrefs for reusable content blocks and keyrefs for reusable variables, showing how a single source of truth can be shared across multiple topics.

## Goals
- Understand content reuse using conrefs
- Understand variable reuse using keyrefs
- Practice modular documentation design

## What this project demonstrates
- Single-sourcing shared content
- Reusing warnings and reference sections
- Separating content from configuration values
